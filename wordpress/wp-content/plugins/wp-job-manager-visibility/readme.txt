Customize listing field visibilities (including output) by specific user or your own custom groups.  Group by user, capability, other custom groups, and WooCommerce packages.

== Description ==

Set Job and Resume field visibilities with custom placeholders based on specific user, or group configurations.  Custom groups can be used to create a complex visibility structure.

Current Group Features:

 * Users
 * Groups (including custom groups)
 * Capabilities
 * Packages ( soon .. )

== Installation ==

= Automatic installation =

Install through Wordpress, select activate.

= Manual installation =

The manual installation method involves downloading the plugin and uploading it to your webserver via your favourite FTP application.

* Download the plugin file to your computer and unzip it
* Using an FTP program, or your hosting control panel, upload the unzipped plugin folder to your WordPress installation's `wp-content/plugins/` directory.
* Activate the plugin from the Plugins menu within the WordPress admin.
