Limit job submissions and resume access through WP Job Manager to paid subscribers in Restrict Content Pro

== Description ==

Limit job submission and / or applications for [WP Job Manger](https://wpjobmanager.com/) to paid subscribers in [Restrict Content Pro](http://restrictcontentpro.com) This extension does NOT function without WP Jobs Manager AND Restrict Content Pro installed.

If you are using the [Resume Manager add-on](https://wpjobmanager.com/add-ons/resume-manager/), this plugin will also limit viewing resume details to paid subscribers.

Learn more about Restrict Content Pro: http://restrictcontentpro.com
